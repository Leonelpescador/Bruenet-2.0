1. Clonar el repositorio
Primero, clona el repositorio desde GitHub a tu computadora local. Abre una terminal y ejecuta el siguiente comando, reemplazando URL_DEL_REPOSITORIO con la URL de tu repositorio:

git clone URL_DEL_REPOSITORIO

2. Navegar al directorio del proyecto
Cambia al directorio del proyecto clonado:

cd E:\Programacion\brunet

3. Crear y activar un entorno virtual
Crea un entorno virtual para tu proyecto. Esto ayuda a mantener las dependencias aisladas:

python -m venv env

Activa el entorno virtual:

E:\Programacion\brunet\env\Scripts\activate

4. Instalar las dependencias
Asegúrate de que el archivo requirements.txt esté en la raíz del proyecto. Luego, instala las dependencias listadas en ese archivo:

pip install -r requirements.txt
